# Instagram Mini Clone

A full-stack Instagram-like web application built to demonstrate modern web development practices.

## Tech Stack
- Frontend: React (Vite), CSS Modules
- Backend: Node.js, Express
- Database: PostgreSQL (Sequelize ORM)
- Authentication: JWT

## Features
- User Signup & Login
- Protected Routes
- Feed with Posts
- Like System (Optimistic UI)
- User Profile Page
- Post Detail with Comments
- Responsive, clean UI

## Run Locally

### Backend
```bash
cd backend
npm install
npm run dev
